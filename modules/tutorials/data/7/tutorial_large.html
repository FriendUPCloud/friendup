Do you want to publish a file on the web?

When going into the Friend file manager, you can right click on an icon, or user the long-press on mobile, and you will be able to see Properties.

In Properties, you can select "Public file". When you do that, you will get a Public url path in the input field below. Click "Save", and you can now copy that public url to a friend, or link to the file on a website.
