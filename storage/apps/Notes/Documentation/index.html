Author - Word processing for FriendUP

Introduction

Author default is the word processing application for FriendUP. With this application, you can write documents and store them in various document formats.

Author is not to be compared with Microsoft Office&team; or Libreoffice. Author has a rather basic feature set, and would compare much better with Wordpad&team; for Microsoft Windows&team;.

Getting started

After you start Author, it displays an empty document screen. Here you can immediately start typing in letters and words. On the main window, you see the pulldown menu indicator. By clicking on this, you will get the menus of Author. Here you can use Save og Load to save or load your document. When selecting either of these, you will be presented with the FriendUP file dialog.
